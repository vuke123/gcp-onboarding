terminalu s postavljenim varijablama.

Added container ID detection using docker compose ps -q pubsub-emulator
Moved all gcloud commands to run inside the container using docker exec
Updated the usage instructions to show the correct commands that use docker exec
Improved error handling by chaining commands with &&
Added better formatting and spacing for readability
Now you can use the script as follows:

To start the emulator and create resources:
bash
./run-lab.sh
To publish a message (using the shown command):
bash
